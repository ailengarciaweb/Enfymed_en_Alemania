# Optimización y Rediseño Web – Enfymed en Alemania
Optimización y rediseño de la web oficial de Enfymed en Alemania. Esta empresa se dedica a dar asesoria en el proceso de homologación de estudios profesionales, principalmente en el área de la salud, en Alemania.
